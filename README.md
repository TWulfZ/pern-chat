# PERN Full-Stack: Chat en tiempo real | Postgres + Express + React + Node.js

![PERN](https://imgur.com/ISPGqoB)

### Para instalar dependencias:

```bash
bun install
```
### Inicializa las variables de entorno .env 

```js
DATABASE_URL=...
JWT_SECRET=...
NODE_ENV=...
PORT=...
```


### Ejecuta el proyecto:

```bash
bun run dev
```
